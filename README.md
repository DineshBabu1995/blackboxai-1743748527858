# RPA Developer Resume

## Project Overview
This project showcases a personal resume for an RPA Developer, highlighting skills, professional experience, projects, and contact information. Built using HTML, CSS (with Tailwind CSS), and JavaScript, this web-based resume is designed to be responsive and visually appealing, leveraging modern web development techniques.

## Installation
To run this project locally, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/rpa-developer-resume.git
   ```
2. Navigate to the project directory:
   ```bash
   cd rpa-developer-resume
   ```
3. Open the `index.html` file in your preferred web browser.

## Usage
- Navigate through different sections of the resume using the navigation bar.
- The resume includes sections such as **About**, **Skills**, **Experience**, **Projects**, and **Contact**.
- You can toggle between light and dark themes using the theme toggle button in the navbar.
- You can fill out and submit the contact form to send a message (currently simulated with an alert box).

## Features
- **Responsive Design:** Adapts to different screen sizes for optimal viewing experiences on mobile and desktop devices.
- **Dark Mode:** Users can switch between light and dark themes.
- **Skill Visualization:** Dynamic skill bars represent proficiency levels in various RPA tools and programming languages.
- **Contact Form:** Users can send messages directly through the website (currently not linked to any backend).
